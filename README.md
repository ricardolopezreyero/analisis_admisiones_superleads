# Análisis admisiones SuperLeads

Tablero de admisiones para cargar archivos de oportunidades (CSV/Excel), generar gráficos ejecutivos y exportar reporte en imagen/PDF.

## Archivo recomendado para producción
- `Analisis_Admisiones_SuperLeads_Para_SuperLeads.html`

Este archivo ya va listo para contenedor (HTML único con CSS y JS integrados).

## Versión modular (desarrollo)
- `index.html`
- `styles.css`
- `app.js`

## Uso rápido
1. Abrir `Analisis_Admisiones_SuperLeads_Para_SuperLeads.html` en navegador.
2. Guardar nombre de institución.
3. Cargar CSV o Excel de oportunidades.
4. Aplicar filtros y descargar CSV/imagen/PDF.
